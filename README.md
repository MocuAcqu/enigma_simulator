在參與台科大一門課程「區塊練與加密貨幣」的期間，老師針對古典密碼學進行了介紹與教學，還帶我們看了2014年的電影《模仿遊戲》，深深的認識了關於 enigma 的歷史背景、原理。

不管是課程中，學習到各種古典密碼學的原理，或是認識那個時代背景，並一起在電影中絞盡腦汁解密，這一切使我覺得興奮與崇拜。在理解各種方法加密解密的邏輯後，都會讓我張開著嘴巴感嘆著究竟是如何發想到這個方法的，其實理論來說，那些作法都不難，甚至以現在的知識水準，導致很容易可以理解背後的思考步驟，但仍然會為能夠提出這些複雜卻優美的思路感到厲害。身於現代的大家，總會覺得創新很難，但在看完圖靈的故事之後，我認為「存在限制的不是世界，而是人本身」，是現代的科技與技術遮蔽了我們對更多可能性的想像與認知。

我們活在了一個資訊龐大的世界，光是嘗試著掌握現有知識都不容易了，而且我們還受到許多環境所賦予的框架與藍圖影響，更無法跳脫思維。而電影裡的圖靈，被戲稱為人群中的異類，雖說是被世人遺棄，但這樣的他卻看見了他人所無法理解的思維，也真正的成就了大事。這讓我不禁想嘗試著，換個心態和眼光看看世界，不要讓自己成為框架的一部分，是否就可以做出其他有趣的事情呢?

除了心理上的體會，我也對於 enigma 的設計也深深著迷，所以我就嘗試設計了這個 Enigma 模擬器 (有使用 Google studio ai 協助加快網站的開發)，了解其中機械結構的原理、加密解密的演算、字母轉換的邏輯，並嘗試讓他可以跟真實的機器一樣做與變換。必須得說，如同遊戲一樣，建構出一個網頁板的成果，十分地有成就感。儘管它無法像是原始機械那樣有著實體的質感 (真的得說，它設計的非常漂亮，在眾多功能下，依然保持精巧與簡易)，但 2D 平面的呈現，也確實別有一般風味，有種童年解謎網站遊戲的感覺，神秘且不失那種簡約的美感。

我對於那個年代使用此科技所造成的災害感到遺憾，對於當時同性戀、女性受到嚴苛待遇也同感不公，雖然戰爭似乎推動了時代的變遷，但我希望未來科技的進步可以不再以犧牲他人為代價，人們可以一同前進，可以一起見證下一個打破限制的世界。
